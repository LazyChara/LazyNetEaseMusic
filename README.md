## ✨ 特性 (Features)

* **HTML5 Audio**: 原生音频支持，无需 Flash 或额外插件。
* **Lazy Design**: 极简 UI 设计，专注于听歌体验，没有多余的社交干扰。
* **Responsive**: 响应式布局，支持 PC 端和移动端访问。
* **Lyric Sync**: 支持 .lrc 歌词同步滚动。
* **Visualizer**: 简单的 CSS 动效或 Canvas 音频可视化。

## 🚀 如何运行 (How to Run)

* 本项目是纯静态网页，无需复杂的后端配置。
* 直接访问web页面即可！
